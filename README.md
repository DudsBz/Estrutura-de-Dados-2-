# Estrutura de Dados 2 — Repositório de Exercícios e Implementações

Este repositório reúne implementações, exercícios e materiais práticos da disciplina "Estrutura de Dados 2". O objetivo é oferecer código comentado e exemplos para estudo, comparação de algoritmos e testes educativos.

## Visão Geral

Este repositório contém implementações típicas de tópicos de um curso de Estruturas de Dados avançado, como árvores balanceadas, listas encadeadas, estruturas lineares (pilhas e filas), algoritmos de busca e ordenação, e exemplos de recursividade.

## Estrutura do Repositório

- `Alocação de Variáveis/`: Exemplos de alocação estática e dinâmica, uso de ponteiros e gerenciamento de memória.
- `Árvores AVL/`: Implementações de árvores AVL, operações de inserção, remoção e balanceamento.
- `Árvores Binárias/`: Árvores binárias de busca, travessias (pré, em, pós-ordem) e utilitários.
- `Classificação de Dados/`: Algoritmos de ordenação (bubble, insertion, quicksort, mergesort etc.) e comparativos básicos.
- `LLDE/`: Lista Ligada Duplamente Encadeada — implementação e operações (inserir, remover, buscar).
- `LLSE/`: Lista Ligada Simplesmente Encadeada — implementação e exemplos de uso.
- `Pesquisa de Dados/`: Algoritmos de busca (linear, binária e variações) e testes de desempenho simples.
- `Pilhas e Filas/`: Implementações de pilhas e filas usando vetores e listas encadeadas.
- `Recursividade/`: Exercícios recursivos; contém arquivos de exemplo como `outro.c`, `positivo_negativo.c` e `potencia.c`.
- `Tuplas/`: Exemplos de estruturas compostas e uso de tuplas/structs para agrupar dados.



## Convenções de Código
- Linguagem: C.
- Comentários claros e funções com nomes descritivos.
- Guardas de inclusão em headers e gerenciamento explícito de memória.
